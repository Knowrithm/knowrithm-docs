# Geographic Data Interactive Menu

The CLI supports interactive menu selections for geographic data operations (countries, states, and cities), making it significantly more user-friendly than manually looking up IDs.

## Interactive Commands

You can run the following commands without arguments to trigger the interactive selection mode:

```bash
# Shows country selection menu
knowrithm system country

# Shows country selection menu, then lists states
knowrithm system states

# Shows state selection menu (via country ‚Üí state)
knowrithm system state

# Shows state selection menu (via country ‚Üí state), then lists cities
knowrithm system cities

# Shows city selection menu (via country ‚Üí state ‚Üí city)
knowrithm system city
```

## Benefits

- **No ID Memorization**: You don't need to know or look up IDs.
- **Guided Navigation**: Natural hierarchical flow (country ‚Üí state ‚Üí city).
- **Visual Feedback**: Emoji indicators and clear prompts guide you.
- **Backward Compatible**: Still accepts IDs directly for scripting/automation.
